# Schema (v1)

Each cell line page should include the sections below (use the same headings for consistency).

## 1. At-a-glance
- Name, RRID, species, tissue, growth mode, recommended medium, BSL

## 2. Identity & provenance
- Provider/catalog, aliases, origin, derivation

## 3. Culture conditions
- Base medium
- Supplements (with final concentrations when sourced)
- Environment (temperature/CO₂/O₂)
- Coating (if required)

## 4. Passaging & routine maintenance
- Dissociation method, split ratio, confluency guidance, media change cadence

## 5. Cryopreservation
- Freezing medium, freezing density (if known), storage

## 6. QC
- Authentication guidance, mycoplasma testing, other notes

## 7. Characteristics
- Morphology, doubling time (with conditions), notable phenotypes

## 8. References
- Vendor page + RRID + key papers

## 9. Data needed
- Unknown or conflicting items to resolve with better sources
