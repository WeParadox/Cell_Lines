# Cell Line Repository

This site collects **culture conditions, handling notes, QC expectations, and references** for commonly used cell lines.

## Principles
- Don’t invent parameters. Mark unknowns as **Unknown**.
- Prefer authoritative sources (ATCC/DSMZ/JCRB/ECACC, RRID, peer-reviewed papers).
- Content is informational; follow institutional SOPs and vendor datasheets.

Use the left navigation to browse cell lines.
